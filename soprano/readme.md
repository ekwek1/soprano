Option,Name,Description
1,API,Launches the standard REST API server (api.py).
2,Test API,Launches api.py and test_api.py with a 10-second delay. It saves the resulting audio inside the /server folder.
3,WebSocket,Launches the WebSocket streaming server for real-time audio.
4,Test WebSocket,"Similar to Option 2, but streams audio in chunks. The test file plays the chunks as they arrive."
5,WebUI,Launches the full Soprano Web Interface.
6,Soprano CLI,Launches the command-line interface for direct interaction.
7,Exit,Closes the menu.
